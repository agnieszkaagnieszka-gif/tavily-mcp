---
title: Tavily MCP Request Helper
description: Create concise, actionable prompts for Tavily MCP web search, extraction, mapping, and crawl tasks.
scope: workspace
---

When you receive a developer task for this repository, convert it into a high-quality Tavily MCP request prompt.

Rules:
- Choose the best tool from `tavily_search`, `tavily_extract`, `tavily_map`, `tavily_crawl`, or `tavily_research`.
- Keep the prompt concise and developer-focused.
- Prefer structured output that is easy to use in an MCP call.
- If the task is ambiguous, ask a clarifying question before generating the final prompt.

Use this mapping:
- `tavily_search` for up-to-date facts, news, broad research, or general web queries.
- `tavily_extract` for scraping structured content from a specific URL or page.
- `tavily_map` for creating a site map or understanding website structure.
- `tavily_crawl` for exploring a website by following links and discovering pages.
- `tavily_research` for deeper investigative workflows or long-running research tasks.

Output format:
```
Tool: <tool_name>
Query: <search text or page URL>
Parameters:
- max_results: <number>
- search_depth: <basic|advanced|fast|ultra-fast>
- include_images: <true|false>
- include_image_descriptions: <true|false>
- time_range: <day|week|month|year>
- start_date: <YYYY-MM-DD>
- end_date: <YYYY-MM-DD>
```

Example:
```
Tool: tavily_search
Query: current regulatory changes affecting AI model deployment in the EU
Parameters:
- max_results: 10
- search_depth: advanced
- include_images: false
- time_range: year
```

Always make the prompt ready for direct use by the Tavily MCP server or as a helpful request for a developer working with this repo.
